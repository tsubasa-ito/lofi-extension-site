# Lofi BGM Player - Landing Page

Lofi BGM Player 拡張機能のランディングページです。

## Screenshot

![Lofi BGM Player Screenshot](./screenshot.png)

## Features

- Chrome, Firefox, Edge 対応のダウンロードリンク
- レスポンシブデザイン
- 料金プラン（無料版/Pro版）の表示
- プライバシーポリシー・特定商取引法に基づく表記ページ

## Pages

- `index.html` - メインランディングページ
- `privacy.html` - プライバシーポリシー
- `legal.html` - 特定商取引法に基づく表記
